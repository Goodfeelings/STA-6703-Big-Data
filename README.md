# STA 6703 - Statistical Learning with Big Data

This repository contains my assignments and projects for the STA 6703 course. 

We use Python 3.10+ and Jupyter notebooks for analysis, utilizing **Quarto** to generate polished PDF reports for submission.

## Project Structure
- **utilities.py**: Contains re-usable functions and helper code to keep notebooks clean.
- **Notebooks/**: Jupyter notebooks (`.ipynb`) for active analysis and modeling.
- **Data/**: Raw and processed datasets (Ignored by Git to manage storage).
- **Outputs/**: Final rendered PDF reports and visualizations.

## Tech Stack
- **Language**: Python 3.10+
- **Environment Management**: [uv](https://github.com/astral-sh/uv)
- **Reporting**: Quarto (PDF export)

## Getting Started
To install the environment and all dependencies, ensure you have `uv` installed and run:
\`\`\`bash
uv sync
